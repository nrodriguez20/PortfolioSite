## Getting Started

1. Clone the repo

git clone https://github.com/nrodriguez20/PortfolioSite.git
cd PortfolioSite

2. Open index.html in your browser.

3. To deploy:

- Push to GitHub
- Go to Settings → Pages → Source: main branch → root to enable GitHub Pages.
